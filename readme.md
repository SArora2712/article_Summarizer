#  Article Summarizer Pro

An AI-powered article summarization system using NLP techniques.

##  Features
- Extractive summarization (TextRank)
- Adjustable summary length
- Clean and simple interface (planned)
- Word importance visualization (planned)

##  Tech Stack
- Python
- NLTK
- Scikit-learn
- NetworkX

##  How It Works
1. Sentence tokenization
2. TF-IDF vectorization
3. Cosine similarity matrix
4. Graph-based TextRank ranking
